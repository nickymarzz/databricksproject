# Security Policy

## Supported Versions

Currently, security updates are only provided for the latest version running on the `main` branch.

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of this project seriously. If you find a security vulnerability, please do NOT open a public issue. Instead, report it privately:

1. Send an email to **<nickymarzz@gmail.com>** with details of the vulnerability.
2. Include steps to reproduce, potential impact, and any suggested fixes or mitigations.
3. We will review your report and respond within 48 hours to coordinate a security release.
